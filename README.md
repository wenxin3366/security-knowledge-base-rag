# security-knowledge-base-rag
基于RAG的网络安全知识库问答助手，支持上传文档并智能回答安全相关问题


项目结构
```txt
rag_cli/
├── main.py              # 主程序入口
├── file_processor.py    # 文件解析和分块
├── vector_store.py      # 向量数据库封装
├── embedding_client.py  # Embedding API 调用
├── chat_client.py       # Chat API 调用
├── config.py            # 配置文件
└── requirements.txt     # 依赖列表
```

组织架构
```txt
┌─────────────────────────────────────────────────────────────┐
│                    命令行 RAG 知识库系统                     │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  启动程序                                                    │
│      ↓                                                      │
│  显示命令菜单                                                │
│      ↓                                                      │
│  等待用户输入命令：                                           │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐    │
│  │ /upload <文件路径>  - 上传文件到知识库               │     │
│  │ /list               - 查看已上传的文件               │    │
│  │ /ask <问题>         - 基于知识库提问                 │     │
│  │ /history            - 查看对话历史                   │    │
│  │ /clear              - 清空对话历史                   │    │
│  │ /delete <文件ID>    - 删除指定文件                   │     │
│  │ /exit               - 退出程序                       │    │
│  └─────────────────────────────────────────────────────┘    │
│                                                             │
│  核心模块：                                                  │
│  ├── 文件解析器 (PDF/TXT/Markdown)                           │
│  ├── 文本分块器 (Chunking)                                   │
│  ├── Embedding 客户端 (调用硅基流动 API)                      │
│  ├── 向量数据库 (Chroma 内存版)                               │
│  ├── 对话历史存储 (内存列表)                                  │
│  └── Chat 客户端 (调用 DeepSeek API)                         │
└─────────────────────────────────────────────────────────────┘
```
